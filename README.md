# Personal_Portfolio

This is a Django-based personal portfolio website.

## Features
- Home page
- About section
- Projects showcase
- Education section
- Contact section

## Technologies Used
- Python
- Django
- HTML, CSS

## How to Run
1. Clone the repository
git clone https://github.com/maricarpordan/personal_portfolio.git
2. Go to the project folder
cd personal_portfolio
3. Create a virtual environment
python -m venv venv
4. Install dependencies:
   pip install -r requirements.txt
5. Run the server:
   python manage.py runserver
